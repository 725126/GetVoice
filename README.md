## 프로젝트 개요
GetVoice는
아마추어 성우와 성우를 필요로 하는 개인·소규모 제작자를 연결하기 위해 기획한 **성우 구인·구직 웹 서비스**입니다.

## 주요 기능
- 구인자 / 구직자(성우) 역할 분리 회원가입 및 권한 기반 메뉴·기능 제공
- 구직자 전용 보이스포트(VoicePort) 포트폴리오 페이지 작성 및 관리 기능
- WebSocket 기반 실시간 채팅(V-Chat)을 통한 의뢰 협의 기능
- V머니 기반 안전거래 흐름 구현 (대본 업로드 → 글자 수 기반 견적 산정 → 거래 확정 후 정산)
- 거래 완료 후 리뷰·평점 시스템을 통한 성우 신뢰도 관리

## 담당 역할 
- 서비스 로직 설계 및 구조 정의 
- 페이지 UI/UX 설계 및 스토리보드 작성
- 데이터베이스 구조 설계 및 테이블 정의
- 프론트엔드 및 백엔드 기능 개발 전반 담당 
- 팀원 작업물 코드 리뷰 및 수정, 최종 통합 및 품질 관리

## 기술 스택
- **OS**: Windows 10
- **Backend**: Java 11, Spring Boot 2.7.9
- **Persistence**: JPA
- **Database**: MySQL (MySQL Workbench)
- **Tool**: IntelliJ IDEA
